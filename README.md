## Getting Started


### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```
* Expo
  ```sh
  npm install expo-cli --global
  ```
* Simulator
https://developer.android.com/studio/run/managing-avds


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/DevilKo0l/delicia-or-delight-app.git
   ```
2. Install expo packages
   ```sh
   yarn add expo
   ```
4. Run on simulator
   ```JS
   yarn start
